# Instantly Campaign Count Launched

Use this tool to get the number of launched Instantly campaigns in the workspace.

Use it when:
- you need a quick count, not campaign records
- you want a cheap smoke test for Instantly connectivity

Arguments:
- none

Returns:
- `count` non-negative integer

Examples:
```json
{}
```

```json
{}
```

Typical use cases:
- check whether a workspace has active launched campaigns
- smoke-test Instantly credentials with a non-destructive read
- compare a workspace before and after campaign launch setup
