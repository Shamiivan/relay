# Public Web Search

Use `public_web_search` for generic public-internet research with read-only web tools.

Available tools:
- `web.search` for result discovery
- `web.fetch` for reading one known URL in full
- `web.fetch-llm` for one-shot multi-source search plus extracted snippets

Use this workflow for broad lookup, fact-finding, and source gathering.
Do not use it for sales prospecting, campaign creation, outreach operations, or any task that needs authenticated/private data.
